Calendário de Conteúdo

Conectar ao Asana

Insira seu Personal Access Token do Asana para acessar os calendários de conteúdo.

Como gerar seu token:
  1. Acesse Asana → My Apps
  2. Clique em "+ Create new token"
  3. Dê um nome (ex: "VDR Calendário") e crie
  4. Copie o token (só aparece uma vez!) e cole abaixo

O token fica salvo apenas no seu navegador (localStorage). Não é enviado para nenhum servidor além do próprio Asana.
